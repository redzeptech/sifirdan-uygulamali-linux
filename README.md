# Sıfırdan Uygulamalı Linux

Bu repo, daha önce hiç Linux kullanmamış birinin terminali korkmadan kullanabilmesi için hazırlanmıştır.

Amaç:
- Komut ezberletmek değil
- Mantığı öğretmek
- Gerçek hayattaki kullanım senaryolarını göstermek

Bu bir okuma rehberi değil.
Bu bir **uygulama rehberi**.

Her bölümde:
- kısa anlatım
- kopyala-yapıştır komutlar
- görevler (tasks)
- mini lab çalışmaları vardır.

---

## Nasıl Kullanılır?

1. VirtualBox veya VMware kur
2. Ubuntu Server indir
3. `START_HERE.md` dosyasından başla
4. Her gün 20-30 dakika ayır

Linux, okuyarak değil **yaparak öğrenilir.**
